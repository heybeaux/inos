# Inos (Inosculate)

A 4D reasoning graph for human + AI collaborative thinking — Ginnung's first user-facing faculty wedge. Think *thinking surface*: nodes, edges, time, and an agent loop participating in-graph. The v1 beachhead is **solo** (a single user with agent assist), not teams; Yjs / multiplayer is a future feature, not the strategic fork. Uses Sonder runtime + Parliament reasoning + Lattice governance + Engram memory in-process — these faculties are load-bearing for the v1 wedge, not Phase-5 nice-to-haves.

**Provides:** thinking-surface, reasoning-graph, collaborative-canvas
**Repo:** https://github.com/heybeaux/inos *(default branch: `master`)*
**Relates to:** Ginnung faculty; embeds Sonder runtime; uses Engram (local) for dedup; uses Parliament for branch-arbitration; uses Lattice for gate policy
